# 📊 Data Visualization Project - Employees Dataset

## 📝 Description
This project is about visualizing employee data using Python. The dataset contains employee details such as ID, Name, Age, City, and Salary. Different charts are used to represent the data in a graphical format for better understanding and analysis.


## ⚙️ Tools Used
- Python  
- Pandas  
- Matplotlib  


## ⭐ Features
- Load employee data from CSV file  
- Analyze employee distribution by city  
- Visualize data using multiple charts:
  - Bar Chart  
  - Pie Chart  
  - Histogram  
  - Line Chart  
- Easy interpretation of salary and city patterns  
- Clear graphical representation of data  


## 📂 Files 
- employee_data.csv 
- data_visualization.py
- data_visualization_screenshot.png 


## 📊 Output
The project generates the following visualizations:
- Bar Chart → Employees count by City  
- Pie Chart → City-wise percentage distribution  
- Histogram → Salary distribution  
- Line Chart → Salary trend based on ID  


## 🎯 Conclusion
This project helps in understanding how data visualization works using Python. It converts raw employee data into meaningful charts, making it easier to analyze salary patterns and city-wise employee distribution.
